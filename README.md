# Gym Training Narrative

Agentic 训练叙事系统 — 将健身训练中的碎片化信号自动拼凑为结构化的训练故事线。

## 架构

```
采集层（碎片捕获器）→ 存储层（Supabase + KV + OSS）→ 展示层（双端 HTML SPA）
```

- **数据库**: Supabase PostgreSQL（训练记录、判断节点、训练主线）
- **KV Store**: PubHTML API（前端实时读写中转）
- **媒体存储**: 阿里云 OSS（视频/图片/录音）
- **中间层**: 阿里云 FC 函数计算（上传签名、OCR、缩略图）
- **前端**: 单文件 HTML SPA，PubHTML 托管

## 核心功能（MVP1）

1. **多模态碎片录入** — 板书照片/视频/录音/语音文字，课中无感采集
2. **Agent 课后拼凑** — 自动合成完整训练记录，只在有歧义时追问
3. **训练主线串联** — 跨多次训练追踪目标进展（如"引体向上从0到1"）
4. **双端协同展示** — 学员端看进展鼓励，教练端看管理交付
5. **媒体时间线** — 视频/图片关联到具体动作，按主线时间线展示
